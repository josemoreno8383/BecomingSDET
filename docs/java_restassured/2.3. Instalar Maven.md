## 🔍 **¿Cómo instalar Maven en Linux Mint?**

### 📌 **Instalación con apt:**

Actualiza la lista de paquetes:


```
sudo apt update
```

Instala Maven:

```
sudo apt install maven
```

Verifica la instalación:

```
mvn -version
```
